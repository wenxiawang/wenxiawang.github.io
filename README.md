## Quick Start

This website is built using the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template.

### Local Development

1. Install Jekyll and dependencies:
```bash
bundle install
```

2. Run the local server:
```bash
bash run_server.sh
```

3. Open http://127.0.0.1:4000 in your browser

### Configuration

- Main configuration: `_config.yml`
- Homepage content: `_pages/about.md`
- Profile image: `images/profile.png`

### Google Scholar Citations

To enable automatic Google Scholar citation updates:

1. Find your Google Scholar ID from your profile URL
2. Add it as a repository secret: `Settings -> Secrets -> Actions -> New repository secret`
   - Name: `GOOGLE_SCHOLAR_ID`
   - Value: Your Google Scholar ID
3. Enable GitHub Actions in the repository
