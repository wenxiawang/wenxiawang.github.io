## 🆘 故障排除

如果遇到问题：

1. **网站无法加载**
   - 检查 GitHub Actions 是否有错误
   - 验证 GitHub Pages 已启用
   - 首次推送后等待 10-15 分钟

2. **样式未显示**
   - 验证 `_config.yml` 中的 `repository` 正确
   - 清除浏览器缓存
   - 检查浏览器控制台错误

3. **图片未加载**
   - 验证文件路径正确
   - 检查文件名完全匹配（区分大小写）
   - 确保图片在 `images/` 目录中

4. **本地服务器无法启动**
   - 重新运行 `bundle install`
   - 检查 Ruby 版本（需要 2.5+）
   - 尝试直接运行 `bundle exec jekyll serve`
